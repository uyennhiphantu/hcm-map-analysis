# HCMC Transportation Infrastructure Analysis (2018-2025)

## Dataset
- **100 random points** across Ho Chi Minh City
- **10,000 OD pairs** (100×100 matrix)
- **Years:** 2018 vs 2025

## Key Findings
- Average improvement: **-47.9 seconds**
- Best improvement: Point 86 (Thu Duc) → **-70 minutes**
- Null routes 2018: 0% | 2025: 3.9%
